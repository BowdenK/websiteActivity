Week 4

This week, I have mostly been learning: The DOM!

Document Object Model (DOM) allows us to manipulate our HTML and CSS code using Javascript scripts, amongst other languages. This can help us turn a rather static webpage into a more dynamic and interactive experience.

A typical set up for accessing DOM objects is to assign them to variables, from there we can call upon them when needed. Properties of objects can be changed, added and removed, giving a wealth of opportunities to make a website more engaging.

I personally found the DOM aspect to be quite intuitive, however keeping track of the many properties and changes, does utilise i good chunk of my personal RAM.

During this week, we were tasked to create drum kit, using what we had learned from our DOM sessions and pulling together our HTML CSS and JS knowledge. This was an enjoyable challenge. I think the greatest hurdles here were the creative side of it.
